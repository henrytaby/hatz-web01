## 1. Diagramas Interactivos con `<Mermaid />`

Podemos escribir diagramas que se renderizan automáticamente y **responden al tema** (claro/oscuro) del sitio web usando Mermaid.js.

<Mermaid chart="graph TD;&#10;  Usuario((Usuario)) --> |Visita /blog| A[Next.js App Router];&#10;  A -->|Ruta estática| B{Cache SSG};&#10;  B -- Hit --> C[HTML Rápido];&#10;  style Usuario fill:#2563eb,stroke:#1d4ed8,color:#fff;" />

**Explicación:** Este diagrama carga de manera asíncrona ("lazy-loaded") para no bloquear la carga inicial. Si cambias el tema en la barra de navegación entre claro y oscuro, el diagrama interceptará el cambio y se redibujará con los colores apropiados al instante.

## 2. Optimización Automática de Imágenes (`next/image`)

Si utilizamos la sintaxis estándar de Markdown para imágenes, nuestro sistema la intercepta y utiliza `<Image />` de Next.js.

**Explicación:** Esta imagen no usa una etiqueta `<img>` tradicional. Por debajo, se ha convertido al formato webp/avif, evita saltos visuales en la página (layout shifts) gracias a su contenedor `aspect-video` y se carga de manera diferida (lazy-loading).

## 3. Syntax Highlighting de Nivel Servidor (`rehype-pretty-code`)

Podemos mostrar código con un resaltado de sintaxis hermoso utilizando **Shiki** en tiempo de compilación.

```typescript
// pages/api/hello.ts
export default function handler(req, res) {
  const message: string = "¡Hola Mundo MDX!";
  console.log(message);
  res.status(200).json({ name: 'Henry Taby' });
}
```

**Explicación:** Este resaltado de colores no requiere cargar pesados scripts de JavaScript en el navegador de tus usuarios; el HTML y las clases CSS ya vienen procesados directamente desde el servidor (Server Components), obteniendo una puntuación perfecta de rendimiento.

## 4. Enlaces Automáticos y Tabla de Contenidos

Si pasas el cursor o haces clic sobre el título de cualquier sección (por ejemplo, el de esta misma sección 4), verás que es un **enlace clickeable**.

**Explicación:** Usando `rehype-slug` inyectamos IDs automáticos en todos los `<h2>` o `<h3>`, y con `rehype-autolink-headings` los hacemos clickeables. Esto es ideal para compartir una sección exacta con otra persona copiando la URL generada en tu navegador (nota cómo el link termina en `#4-enlaces-automáticos-y-tabla-de-contenidos`).
